青岛给我的感觉像是放大版的厦门。比厦门大气，但是不如厦门精致。

<strong>环岛路</strong>

海滨城市大抵都有环岛路。但是当我看到青岛地图时，还是忍不住发出了惊叹。竟然也有音乐广场！竟然也有海韵园！看到这些熟悉的名字，从心底涌上一股莫名的感觉，五味杂陈。

先前设计的路线是从栈桥开始往东走。刚下公交车，扑面而来就是浓烈的海腥味，于是身体里原先沉睡的某些东西被唤醒了。本科时虽然无数次地从白城走过，却从来没觉得海的味道这么让人怀念。青岛的海岩很像那种火山地貌，远看有点恶心，走近玩味，变化十足。

与环岛路平行的一些巷子里，很多当地人兜售小海产，贝壳、海星制品琳琅满目，而且可以砍价，比海边便宜不少。

<a href="http://www.yupoo.com/photos/vaneswh/81433123/" title="IMG_1269"><img src="http://pic.yupoo.com/vaneswh/B8ygPrEE/medium.jpg" alt="路边的干货摊子" width="500" height="375" border="0" /></a>

<a href="http://www.yupoo.com/photos/vaneswh/81433125/" title="IMG_1271"><img src="http://pic.yupoo.com/vaneswh/B8yh0l6G/medium.jpg" alt="传说中的莲花果" width="500" height="375" border="0" /></a>

青岛的海滨浴场比我想象中的要差了一截。原本以为厦门海滨开发已经做得够差劲了，现在发现其实各个地方都是半斤八两。第一浴场倒是有很多游乐项目，摩托艇、海上脚踏车之类，看上去都很有趣，不过价钱也高得吓人。

<a href="http://www.yupoo.com/photos/vaneswh/81434395/" title="IMG_1308"><img src="http://pic.yupoo.com/vaneswh/B8yP8TyZ/medium.jpg" alt="可惜天气不给力" width="500" height="375" border="0" /></a>

沿着浴场往前走就到了八大关。研究别人的攻略时，大部分人都强烈推荐这个地方。所谓八大关其实是由八条路交错围城的一个区域，全部坐落着殖民时期的各国建筑，别有一番风情。想来厦门的鼓浪屿也号称“万国建筑博览园”来着，和八大关相比，多的是小资情调，但我倒是更喜欢八大关的植物，可能与季节有关，各种花草争奇斗艳，煞是感人。

<a href="http://www.yupoo.com/photos/vaneswh/81434404/" title="IMG_1317"><img src="http://pic.yupoo.com/vaneswh/B8yPzhT0/medium.jpg" alt="山海关路" width="500" height="375" border="0" /></a>

<a href="http://www.yupoo.com/photos/vaneswh/81434424/" title="IMG_1343"><img src="http://pic.yupoo.com/vaneswh/B8yQnJPs/medium.jpg" alt="秒速5厘米？！" width="375" height="500" border="0" /></a>

要说青岛缺少情调，也是有失公允的。这不，我在八大关的路上发现了“大学之道，在止于至善”的涂鸦。奇怪，青岛人怎会别出心裁地在这里写上厦大校训的出处呢？看来世间处处有美丽的巧合:)

<a href="http://www.yupoo.com/photos/vaneswh/81434435/" title="IMG_1346"><img src="http://pic.yupoo.com/vaneswh/B8yQQpYs/medium.jpg" alt="大学之道" width="500" height="375" border="0" /></a>

<a href="http://www.yupoo.com/photos/vaneswh/81434443/" title="IMG_1347"><img src="http://pic.yupoo.com/vaneswh/B8yRbGt6/medium.jpg" alt="住在这小别墅里一定很爽..." width="500" height="375" border="0" /></a>

一路上的音乐广场、五四广场都是市民散步纳凉的好去处，这两者之间有一个旅游产品购物街。不过不推荐在这里买特产，明摆着是坑人的。走过路过，看看就好。

<a href="http://www.yupoo.com/photos/vaneswh/81434483/" title="P1020940"><img src="http://pic.yupoo.com/vaneswh/B8ySMSJj/medium.jpg" alt="前进前进前进！" width="375" height="500" border="0" /></a>

原计划到奥帆博物馆一游，但时间有限，在帆船基地附近的大火炬下合影之后，我们就打道回府了。

<a href="http://www.yupoo.com/photos/vaneswh/81434489/" title="P1020971"><img src="http://pic.yupoo.com/vaneswh/B8yTbpVM/medium.jpg" alt="祥云在身后..." width="375" height="500" border="0" /></a>

<strong>啤酒街</strong>

青岛打的旅游牌之一就是特色文化街，其中三条：婚纱街、啤酒街、纺织街相距不远。啤酒街上坐落着鼎鼎大名的青岛啤酒厂。参观门票每人50RMB，在A展馆可以看到啤酒厂的百年历史，在B展馆可以直击啤酒制作流程，参观运作中的车间，还可以免费品尝到原浆和纯生啤酒。

<a href="http://www.yupoo.com/photos/vaneswh/81434765/" title="IMG_1358"><img src="http://pic.yupoo.com/vaneswh/B8z1Zx2N/medium.jpg" alt="啤酒博物馆" width="500" height="375" border="0" /></a>

<a href="http://www.yupoo.com/photos/vaneswh/81434787/" title="IMG_1413"><img src="http://pic.yupoo.com/vaneswh/B8z2sF8E/medium.jpg" alt="赠酒" width="375" height="500" border="0" /></a>

啤酒街的整个设计就是围绕着啤酒展开的，所以在包括路灯、休息长椅、窨井盖在内的各种设施上都能看到啤酒的影子。街上最多的就是海鲜馆，在七八月份的晚上，海鲜正盛之时，佐以生啤品尝，那滋味一定惬意。

<a href="http://www.yupoo.com/photos/vaneswh/81434725/" title="IMG_1349"><img src="http://pic.yupoo.com/vaneswh/B8z1yh6O/medium.jpg" alt="老虎也爱喝啤酒" width="500" height="375" border="0" /></a>

<strong>劈柴院</strong>

劈柴院位于中山路，其实就是一个戏楼子。想以前估计有不少人光顾，听听小曲、品品茶、吃吃饭。现在作为一个旅游景点，自然在周围衍生出一系列吃喝之地，鳞次栉比，五花八门，卖什么的都有。

<a href="http://www.yupoo.com/photos/vaneswh/81434458/" title="IMG_1422"><img src="http://pic.yupoo.com/vaneswh/B8yRPKX2/medium.jpg" alt="劈柴院" width="500" height="375" border="0" /></a>

<strong>教堂</strong>

老市中心有两个比较有名的教堂，天主教堂不开放，基督教堂门票每人7RMB，可以参观钟楼。虽然天主教堂不开放，但丝毫不影响人气。仅在我们休息的十多分钟内，就能看到十几对情侣在附近拍婚纱照。果然现代人都热爱异国情调啊...

<a href="http://www.yupoo.com/photos/vaneswh/81434814/" title="IMG_1433"><img src="http://pic.yupoo.com/vaneswh/B8z3l76p/medium.jpg" alt="好多好多婚纱" width="375" height="500" border="0" /></a>

<a href="http://www.yupoo.com/photos/vaneswh/81434800/" title="IMG_1432"><img src="http://pic.yupoo.com/vaneswh/B8z2Qb0C/medium.jpg" alt="我是花瓣控" width="500" height="375" border="0" /></a>

基督教堂现在仍在使用中，每周日上午做礼拜。钟楼里的机械钟在见证了100年的风风雨雨后的今天仍在运转，也只有在这样的古物面前，你可以感受到时间和空间的维度，感受到真切存在过的历史。

<a href="http://www.yupoo.com/photos/vaneswh/81434831/" title="IMG_1440"><img src="http://pic.yupoo.com/vaneswh/B8z3QU7y/medium.jpg" alt="基督教堂在江苏路" width="500" height="375" border="0" /></a>

匆匆的青岛之行一晃而过，原先很多计划去的地方最后都没去成。花两天去了解一个城市实在太短，但拥有一份亲身体验已足够宝贵。只企盼工作之后，还能够有这样的时间和心情，去往更多的地方，感受不一样的风土人情。
